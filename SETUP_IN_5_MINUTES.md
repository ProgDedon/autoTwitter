## Troubleshooting

### "Account not verified" Error
- Make sure you copied the credentials correctly
- Verify your credentials are from Step 1
- Click the Verify button again

### "403 Forbidden" Error  
- You need **Read and Write** permissions
- Go to developer.twitter.com ‚Üí App Settings ‚Üí Permissions
- Change from "Read" to "Read and Write"
- Save and try again

### "Token is invalid" Error
- Copy fresh credentials from developer.twitter.com
- Make sure no extra spaces are pasted
- Try verifying again

### Still not working?
1. Check the browser console (F12 ‚Üí Console tab) for error messages
2. Check the terminal where you ran `npm start` for server errors
3. Make sure ports 3000 and 3001 are available
4. Try restarting: `Ctrl+C` then `npm start`
